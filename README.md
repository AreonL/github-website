# How to build/deploy

## Local

`npm start`

## To github

Input this into the terminal:

`export NODE_OPTIONS=--openssl-legacy-provider`

Then deploy it

`npm run deploy`

## Webpage url

Head to the page: [Webpage](https://areonl.github.io/github-website)

## Commit to github

```bash
git add .
git commit -m "<message>"
git push
```
